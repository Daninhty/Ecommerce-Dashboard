# 📊 Product & Customer Analytics Dashboard

## 📌 Overview
This project presents an end-to-end e-commerce data analysis using Power BI, focusing on product performance, customer segmentation (RFM) and retention analysis. The goal is to generate actionable insights to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze product sales performance  
- Segment customers based on behavior (RFM)  
- Measure customer retention  
- Identify business growth opportunities  

---

## ⚙️ Tools & Technologies
- Microsoft Excel → Data Cleaning  
- SQL Server → Data Transformation & Analysis  
- Power BI → Data Visualization  

---

## 📊 Key Analysis
### 📦 Product Analysis
- Top Products by Revenue & Quantity  

### 👥 Customer Analysis (RFM)
- Segmentation based on:
  - Recency  
  - Frequency  
  - Monetary  

### 🔁 Cohort / Retention Analysis
- Customer retention tracking over time  

### 📈 Time Analysis
- Monthly Revenue Trends  
- Month-over-Month (MoM) Growth  

## 🚀 How to Use
- Download the .pbix file from repository
- Open using Power BI Desktop
- Explore filters (Year, Hotel)
- Interact with dashboard visuals
